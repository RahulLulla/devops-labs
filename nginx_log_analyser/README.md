# nginx_log_analyser

## Getting Started

1. **Clone Repo**

```
git clone https://github.com/RahulLulla/devops-labs.git
cd nginx_log_analyser
```

2. **Make Script executable and run it**

```
chmod +x nginx_analyser.sh
sh nginx_analyser.sh
```
